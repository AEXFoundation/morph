# Morph

A console-based image processing engine that uses nodes to apply and combine filters, enabling complex image transformations directly from the command line.

## Features

**CLI-First**  
Seamlessly integrate into any workflow or automation script

**Python Library**  
Use Morph directly in your Python scripts and notebooks

**Extensible**  
Designed to support a vast collection of composable image filters

**Universal**  
Works with any application or development environment
